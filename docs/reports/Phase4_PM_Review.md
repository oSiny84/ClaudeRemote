## 2. PM 결정사항

### 결정: Cowork 모드의 프로젝트 섹션 → 표시하지 않음
- Cowork 모드는 Chat과 Code의 중간 성격
- 현재 Claude 앱에서 Cowork 모드의 프로젝트 관련 동작이 불명확
- Code 모드에서만 프로젝트 섹션 표시 (현재 구현 유지)
- 추후 Cowork 모드 동작이 명확해지면 재검토
